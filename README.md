# MirkoC-Porsjekt


USART/PC USB
  - Send generic data via USB to PC

Timer
  - Polling shuffle for ADC data (each sensor gets 5s window)

ADC
  - Potentiometer
        - Adjust power for heating unit
  - Temperature sensor
        - Feedback
  - Light sensor
        - Time-of-day sensor to regulate boost for nighttime heating

Interrupt
  - Start/stop heating unit
